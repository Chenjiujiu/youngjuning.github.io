这是一种非常方便的操作方法，特别是在需要批量添加字符的时候。以下是这个方法的详细步骤：

1. 首先，在 Mac 上打开文本编辑器或其他编程工具。
2. 选中你想要添加字符的多行文本。
3. 按住 {% kbd shift %} 和 {% kbd option %} 键，并用鼠标点击其中一行的行首或行尾。
4. 这样就可以在所有选中的行的行首或行尾添加同样的字符了。

{% kbd shift %} 和 {% kbd option %} 键需要同时按住，且在点击行首或行尾之前不要释放它们。
