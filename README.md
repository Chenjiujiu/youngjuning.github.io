## 克隆

```sh
git clone https://github.com/youngjuning/youngjuning.github.io.git --recurse-submodules
```

## 更新

```sh
git submodule update --init --recursive
```

## 安装依赖

```sh
yarn install
```

## 运行

```sh
yarn server
```
